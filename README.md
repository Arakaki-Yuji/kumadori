# Kumadori

This is simple decorator for Ruby.

Kumadori mean make-up for Japanese Kabuki.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'kumadori'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install kumadori
